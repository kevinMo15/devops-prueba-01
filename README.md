# Clon de Netflix

Este proyecto es un clon estático del sitio web de Netflix. Proporciona a los usuarios una representación visual de las características y funcionalidades clave de la plataforma Netflix. Los usuarios pueden ver el diseño, el diseño y la estructura del contenido del sitio web de Netflix.

![](./img.png)

## Estructura del Proyecto

- **Archivo HTML**: El archivo `index.html` define la estructura del clon de Netflix, incluyendo la navegación, la sección principal, las tarjetas de contenido, la sección de preguntas frecuentes y el pie de página.
- **Archivo CSS**: El archivo `style.css` contiene estilos para replicar la apariencia visual del sitio web de Netflix, incluyendo el diseño, la tipografía, los colores y la adaptabilidad.

## Instrucciones para Ver

Para ver el clon de Netflix:

1. Abre el archivo `index.html` en cualquier navegador web.
2. Explora las diferentes secciones del sitio web, incluyendo la navegación, el contenido destacado, las preguntas frecuentes y los enlaces del pie de página.
3. Toma nota del diseño, los elementos visuales y la presentación del contenido para entender cómo es el sitio web de Netflix.
